# 👔 每日天气穿搭助手

根据天气情况智能推荐穿搭方案，男女分版，覆盖多种场景。

## 功能

- 🌤 **天气获取**：自动获取当前位置天气
- 👕 **智能推荐**：基于温度、天气状况推荐穿搭
- 👫 **男女分版**：独立界面和推荐逻辑
- 📅 **多日预报**：查看未来几天穿搭建议

## 快速开始

打开页面 → 允许定位 → 查看穿搭推荐

## 技术栈

- 前端: 原生 HTML5 / CSS3 / JavaScript
- 部署: GitHub Pages
- 天气API对接
